# 우진만세 예약 프로토타입 — 개발 핸드오프

## 무엇인가
반려견 미용샵 "우진만세"의 **매장 첫 화면 → 예약 플로우** 클릭형 프로토타입입니다.
디자인·인터랙션 의도가 모두 동작하는 형태로 구현돼 있으니, **이 동작과 디자인을 그대로 재현**하는 것을 목표로 해주세요.

## 파일 구조 (구조 그대로 유지)
```
예약 프로토타입.html      ← 셸: 모든 CSS + React/Babel 로드 + 스크립트 마운트
proto/icons.jsx          ← SVG 아이콘 컴포넌트 (★ 일부 교체 예정 — 아래 참고)
proto/screens.jsx        ← 화면 4개: Home / Booking / Info / Done
proto/app.jsx            ← 상태·라우팅 + 더미 데이터
매장 첫화면 통합안 v6.html  ← 홈 화면 "디자인 확정본" (정적, 참고용 기준)
```

## 화면 흐름
1. **Home** — 갤러리(스와이프) + 프로필 + 소셜/영업시간 + 요금·오늘 예약 시간 + 하단 `예약하기`
2. **Booking** — 시술 선택 → 날짜 띠(일요일 휴무 비활성) → 시간 그리드(마감 시간 취소선). 날짜+시간 모두 선택해야 `다음` 활성화
3. **Info** — 반려견 정보 입력 + 요청사항 + **쿠폰 토글**(켜면 -10,000원 즉시 반영) + 현장 결제 금액 요약
4. **Done** — 완료 + 예약 요약 카드

## 동작하는 인터랙션 (반드시 유지)
- 영업 중 `▾` 탭 → 요일별 영업시간 펼침
- 갤러리 도트 이동, 시술/날짜/시간 선택 상태 표시
- 쿠폰 on/off → 결제 금액 실시간 재계산
- 화면 전환 슬라이드 + 뒤로가기

## 데이터 (지금은 더미 → API로 교체)
`proto/app.jsx` 상단에 모두 모여 있습니다. 이 부분을 실제 API 응답으로 갈아끼우면 됩니다.
- `SERVICES` — 시술 목록(id, name, dur, desc, price)
- `DATES` — 예약 가능 날짜(off = 휴무)
- `TIMES` / `BOOKED` — 시간 슬롯 / 마감된 시간
- `COUPON` — 쿠폰 할인액
- `fmt` — 통화 포맷

## ★ 아이콘 교체 안내 (중요)
현재 `proto/icons.jsx`의 **인스타그램 / 쓰레드 / 카카오톡** 아이콘은 임시 SVG입니다.
→ **별도로 정식 아이콘(파일)을 전달할 예정**이니, 받는 즉시 `Ig`, `Threads`, `Kakao`
   세 컴포넌트를 전달받은 에셋으로 교체해주세요. (나머지 아이콘 Pin/Chat/Back/Check/Ticket은 유지)
   - 자리·크기: 홈 화면 소셜 칩은 42×42px, 칩 안 아이콘은 흰색/카톡은 진한 갈색입니다.

## 실행
별도 빌드 없이 `예약 프로토타입.html`을 브라우저로 열면 됩니다.
React 18.3.1 + Babel standalone을 **CDN(unpkg)**에서 불러오므로 **인터넷 연결 필요**.
실서비스로 옮길 때는 인라인 Babel 대신 사전 컴파일(Vite/Next 등) 권장.
