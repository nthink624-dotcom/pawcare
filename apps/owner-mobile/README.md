# PetManager Owner Mobile Shell

This app is a separate Capacitor shell that loads the existing owner admin web URL inside a native iOS/Android container.

The main web app under the repo root is intentionally untouched. Set `OWNER_WEB_URL` in a local `.env` file to point at the existing owner admin web page when you are ready to test.

See [docs/run-build.md](./docs/run-build.md) for setup, sync, and build steps.
