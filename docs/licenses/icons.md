# PetManager Icon License Notes

## Untitled UI Icons

- Source: https://www.figma.com/community/file/1114001199549197320
- Official icon page: https://www.untitledui.com/free-icons
- Official license page: https://www.untitledui.com/license
- Intended usage in PetManager: product UI icons only.

## Usage Summary

Untitled UI states that the free icon set can be used in unlimited personal and commercial projects without attribution.

Do not sell, sublicense, redistribute, or package the original icon files as a separate icon library, UI kit, template, or asset product.

## PetManager Internal Rule

Only add the SVG icons that are actually used by the product.

Recommended location for exported icons:

```text
public/icons/untitled-ui/
```

When adding icons, keep the original source link in this file so the source and license decision remain traceable.

## Notes

The Figma Community file should still be checked visually before bulk export, because individual community pages can show additional creator notes or file-specific terms.
