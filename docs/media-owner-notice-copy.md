# Media Owner Notice Copy

PetManager should clearly explain that message photos are temporary unless the owner saves them into a grooming record.

## Owner Upload Notice

Use near the photo upload button or send modal.

```text
전송용 사진은 30일 동안만 보관돼요.
오래 보관해야 하는 사진은 미용기록에 저장해 주세요.
```

Short version:

```text
전송용 사진은 30일 후 자동 정리돼요.
```

## Before Send Confirmation

Use when the owner sends before/after photos to a customer.

```text
고객에게 보낸 사진은 최근 전송 내역에서 30일 동안 확인할 수 있어요.
미용기록에 저장한 사진은 계속 보관됩니다.
```

## Grooming Record Save Notice

Use when the owner chooses whether to save a photo into the record.

```text
이 사진을 미용기록에 저장하면 고객 상세에서 계속 확인할 수 있어요.
저장하지 않은 전송용 사진은 30일 후 자동 정리됩니다.
```

## Usage Warning

Use when the shop approaches the monthly soft limit.

```text
이번 달 사진 사용량이 많아지고 있어요.
전송용 사진은 자동 압축되며, 30일 후 자동 정리됩니다.
```

When over the soft limit:

```text
이번 달 사진 사용량이 권장 기준을 넘었어요.
사진 전송은 계속 가능하지만, 장기 보관은 미용기록 저장 사진 위주로 관리해 주세요.
```

## Compression Explanation

Use when the owner is worried about image quality.

```text
사진은 모바일 확인에 적합한 크기로 자동 최적화돼요.
원본보다 용량은 줄지만, 고객이 휴대폰으로 확인하기에는 충분한 품질을 유지합니다.
```

Short version:

```text
사진은 전송 전에 자동 최적화돼요.
```

## Customer-Facing Copy

Keep this minimal. Customers do not need storage-policy detail unless they are viewing a photo link.

```text
사진 링크는 일정 기간 후 만료될 수 있어요.
필요한 사진은 기기에 저장해 주세요.
```

## Policy Summary For Settings

Use in a future settings/help surface.

```text
전송용 사진: 30일 보관
미용기록 저장 사진: 계속 보관
원본 사진: 기본 보관 안 함
```

## Tone Rules

- Do not say "삭제됩니다" as the main phrase unless legally required.
- Prefer "자동 정리돼요" because it feels less alarming.
- Always distinguish "전송용 사진" from "미용기록 저장 사진".
- Make it clear that important photos should be saved to the grooming record.
