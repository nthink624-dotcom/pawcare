## 이번 수정 범위 요약

### 실제 수정한 항목

1. 예약 조회 탭의 `예약 추가`
   - 헤더 우측 상단에서 제거
   - 예약 조회 섹션 하단 full-width CTA로 이동

2. 고객관리 탭의 `고객 추가`
   - 헤더 우측 상단에서 제거
   - 고객 목록 섹션 하단 full-width CTA로 이동

3. 고객 상세 화면의 뒤로가기
   - 좌측 상단 텍스트 링크에서
   - 좌측 상단 back button 스타일로 정리
   - 기존 onClick 유지

### 보류한 항목

- 고객 상세 패널 헤더의 `편집/저장`
  - 하위 상세 내부 편집 상태와 직접 연결되어 있어, 지금 단계에서 하단 CTA로 내리면 저장 인지 흐름이 바뀔 수 있음
- 고객 메모 블록 내부 `저장/취소`
  - 인라인 편집 블록이라 전역 CTA 규칙으로 강제 이동하면 오히려 사용성이 나빠질 수 있음
- 반려동물 탭의 `+ 아기 추가`
  - 전역 메인 CTA가 아니라 탭 내부 보조 추가 액션이므로 유지
